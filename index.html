Manage and annotate your scientific data and publish as open data!

Annotate

Add rich fully structured, treelike metadata to your data to give them more meaning

Find

Organize data and make the data findable and browsable via a rich facet structure

Publish

Publish your data and make it citable via a digital object identifier (DOI), thus linking data to publications

Implement FAIR data policy

SciCat is an important building block that allows you implement a FAIR data managmement policy more easily (Image by Gerd Altmann from Pixabay)

Automate your Workflows

Use message passing based on RabbitMQ or Kafka for workflow automation , e.g. for archive integration (Image by Gerd Altmann from Pixabay)

Integrate and Extend

Easily extend system by exploiting the powerful REST based Open API. Using Swagger, the API is well documented and flexible. Clients can be generated with a single command for Python, JS, Java, Go and more (Image by ar130405 from Pixabay)

Modern Techonolgy Stack

Based on modern technology stack Angular 9+ for the frontend GUI and Loopback 3 and MongoDB for the backend to allow a flexible and reliable way to store, manage, and publish data ( Image by Alexandra_Koch from Pixabay )

Open Source Community

Fully Open Source with a committed developer team. Submit PRs on Github. (Photo by Tim Mossholder on Unsplash)
